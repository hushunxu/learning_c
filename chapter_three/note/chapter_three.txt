3.4 switch语句

switch (表达式) {
  case 常量表达式 : 语句
     break;
  case 常量表达式 : 语句
     break;
  default : 语句
}

3.5 while循环与for循环

while (表达式) {
  语句
}


for (表达式1; 表达式2; 表达式3) {
  语句
}

for循环等价于
表达式1;
while (表达式2) {
  语句
  表达式3;
}

3.6 do-while循环

do
  语句
while (表达式);

先执行语句，再判断表达式
do-while至少需要执行一次


3.7 break语句与continue语句

break语句：从for/while/do-while等循环中提前退出
continue语句：用于跳出for/while/do-while本次循环

区别：break跳出整个循环，continue跳出本次循环

3.8 goto语句与标号

goto用法：终止程序在某些深度嵌套的结构中的处理过程，例如一次跳出两层或多层循环

标号：命名与变量命名形式相同，后面要紧跟一个冒号。可以位于对应的goto语句所在函数的任何语句的前面，作用域是整个函数。

例：
     for (i = 0; i < n; i++) {
        for (j = 0; j < m; j++) {
            if (a[i] == b[j]) {
                goto found;
            }
        }
     }
     found :
goto和标号不易维护，慎用。
